== What is this?
 Empty Framework 5 (layout) takes the basic structure of HTML documents in Empty Framework 5. This means that you can write you content once and then just switch you CSS file to have an iPhone or Desktop layout.

== Who is this for?
 This is for people that want to support having a basic structure and implement the same skeleton on their website.

== Where do I begin!?
 Check out the following files and view their source
  html/skeleton.html (uses /html/css/desktop.css)
  html/css/skeleton.css
